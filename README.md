## Ejercicio 2 Taller Android

Desarrollar con el IDE oficial Android Studio un proyecto que presente los siguientes elementos:

Una aplicación que se conecte a un servicio para obtener un catálogo de productos. Los productos se deberán presentar en un listado con un ListView, RecyclerView o un contenedor (ViewGroup) similar.


Se debe crear también un diseño para cada elemento de la lista definido por el programador, tomando en cuenta los elementos que devuelve la conexión.
